FRONTEND.GUIDE - Стиль семейного древа

ОБЩАЯ КОНЦЕПЦИЯ
Тёплый нейроморфизм (Warm Neumorphism) - мягкий, тактильный, "бархатный" интерфейс в бежево-карамельной гамме. Все элементы выглядят как физические объекты на однородной поверхности.

ЦВЕТОВАЯ ПАЛИТРА (обязательно)

Фон страницы - #F5EDE3 (тёплый кремовый)
Фон карточек и блоков - #E8DDD0 (светлый беж)
Основной акцент для кнопок и иконок - #D4BBA5 (карамельный)
Основной текст - #5C4E3D (тёмно-шоколадный)
Второстепенный текст - #8E7A68 (кофейный)
Светлая часть теней - #FFFFFF
Тёмная часть теней - #C9BDB0

ЦВЕТА СОСТОЯНИЙ (альтернативы ярким цветам)

Ошибка: #5C4E3D (тёмный текст) + иконка предупреждения из react-icons
Успех: #8E7A68 (кофейный) + иконка галочки из react-icons
Предупреждение: #D4BBA5 (карамельный)
Ссылки: #D4BBA5, при наведении #5C4E3D
Активный элемент: bg-#D4BBA5 text-white (карамельный фон)

ПРАВИЛА ТЕНЕЙ (главное в нейроморфизме)

Для выпуклых элементов (карточки, кнопки, аватарки):
box-shadow: 8px 8px 16px #C9BDB0, -8px -8px 16px #FFFFFF;
border-radius: 20px;

Для вогнутых элементов (поля ввода, активные или нажатые кнопки):
box-shadow: inset 4px 4px 8px #C9BDB0, inset -4px -4px 8px #FFFFFF;
border-radius: 20px;

Для эффекта при наведении на карточку (приподнимается):
box-shadow: 12px 12px 24px #C9BDB0, -12px -12px 24px #FFFFFF;
transform: translateY(-4px);
transition: all 0.3s ease;

КАРТОЧКИ ЛЮДЕЙ

Каждая карточка должна быть выпуклой с использованием первого правила теней.
Аватарка должна быть круглой, обёрнутой в выпуклый круг с такой же тенью.
Внутри карточки: имя жирным шрифтом цветом #5C4E3D, даты жизни цветом #8E7A68, маленькая иконка рода.
При наведении карточка приподнимается с использованием hover-эффекта.
При клике на карточку открывается модальное окно с полным профилем человека.

ТИПОГРАФИКА

Заголовки H1-H3: жирный шрифт (700), размер 28-22 пикселя, цвет #4A3F33.
Основной текст: обычный шрифт (400), размер 16 пикселей, цвет #5C4E3D.
Второстепенный текст: обычный шрифт (400), размер 14 пикселей, цвет #8E7A68.
Межстрочный интервал: 1.6.
Шрифт: системный (-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif), без засечек.

БИБЛИОТЕКИ (установить обязательно)

npm install react-icons @headlessui/react

react-icons - для иконок (пол, редактирование, поиск, стрелки)
@headlessui/react - для модальных окон (современная библиотека с хорошей accessibility: фокус-трэппинг, Escape, aria-атрибуты)

ОТСТУПЫ И КОМПОНОВКА

Внутренние отступы карточек: padding 24px 28px.
Отступы между карточками в сетке: gap 30px.
Главный контейнер: max-width 1280px, выравнивание по центру.
Схема поколений: выравнивание по центру, линии-соединители цветом #D4BBA5, толщина 2 пикселя, скруглённые углы.

АДАПТИВНОСТЬ

Десктоп (>1024px):
- Главный контейнер: max-width 1280px
- Карточки в сетке: 3 колонки
- Боковая панель фильтров: видна

Планшет (768-1024px):
- Главный контейнер: max-width 100%, padding 16px
- Карточки в сетке: 2 колонки
- Боковая панель: сворачивается в выпадающее меню

Мобильный (<768px):
- Главный контейнер: max-width 100%, padding 12px
- Карточки в сетке: 1 колонка (stacked)
- Боковая панель: модальное окно или сворачиваемый блок
- Кнопки: полная ширина
- Навигация: бургер-меню или горизонтальный скролл

ЧЕГО НЕЛЬЗЯ ДЕЛАТЬ

Запрещены яркие цвета (красный, синий, зелёный) - только бежевая гамма.
Запрещены яркие цвета для состояний (ошибка/успех) - используйте иконки + текст.
Запрещены резкие тени с использованием черного цвета - только мягкие бежевые тени.
Запрещены прямые углы - всегда скругления от 12 до 40 пикселей.
Запрещена прозрачность и стеклянные эффекты - это другой стиль, здесь поверхность плотная.

КОНТРОЛЬНЫЙ ЧЕК-ЛИСТ ДЛЯ АГЕНТА

Перед сдачей проверь:

Все карточки имеют выпуклые тени

Фон страницы - #F5EDE3

Все скругления от 12 пикселей и выше

Иконки из react-icons, карамельного цвета #D4BBA5

При клике на карточку открывается модальное окно

Шрифт системный, без засечек

Адаптивность: проверить на 320px, 768px, 1024px, 1280px
